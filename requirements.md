# Game Application

## User use-cases

### Register

User can register to application by phone number

### Login

User can log in to the application by phone number and password

## Game use-cases

### Each game have a given number of questions
### The difficulty level of questions are "easy, medium, hard"
### Game winner is determind by number of correct answers that each user answered
### Each game is belonged to specific category: sport, history, etc

## Entities

### User

- ID
- Phone number
- Avatar
- Name

### Game

- Id
- Category
- Question List
- Players
- Winners

### Question

- ID
- Category
- Question
- Answer List
- Correct Answer
- Difficulty
